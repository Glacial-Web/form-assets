# 123Form Assets
Assets, mainly CSS, to be unsed to on forms. Hosted in GitHub to make use of jsDelivr CDN.

## Usage
### Branch
Ex: use the lasik-self-test.css - use uri: `https://cdn.jsdelivr.net/gh/Glacial-Web/form-assets@main/lasik-self-tests/lasik-self-test.css`
@main is the branch name, can be changed to a specific version if needed.

### Tag
You can also use tags instead of branch names for versioning.
Ex: use a tag instead of branch - use uri: `https://cdn.jsdelivr.net/gh/Glacial-Web/form-assets@v1.0.0/lasik-self-tests/lasik-self-test.css`

### Latest
Use the latest tag by using `@latest` instead of branch name or specific tag.
Ex: use the latest tag - use uri: `https://cdn.jsdelivr.net/gh/Glacial-Web/form-assets@latest/lasik-self-tests/lasik-self-test.css`
